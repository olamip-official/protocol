El Protocolo Abierto de Lenguaje Alineado y Máquina‑Interpretables (OLAMIP) define un formato de archivo abierto e interoperable para representar datos lingüísticos, morfológicos y fonológicos de manera estructurada y legible por máquinas.

Este repositorio incluye:

Documentación legible para humanos (/docs)

Esquemas legibles por máquinas (/specs)

Archivos de ejemplo OLAMIP (/examples)

Flujos de trabajo de integración continua (/.github/workflows)

Primeros Pasos
Consulta docs/overview.md para una introducción general.
