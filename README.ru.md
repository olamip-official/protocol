Открытый протокол языкового выравнивания и машинной интерпретации (OLAMIP) определяет открытый, совместимый формат файлов для представления лингвистических, морфологических и фонологических данных в структурированном, машиночитаемом виде.

Этот репозиторий включает:

Документацию для людей (/docs)

Машиночитаемые схемы (/specs)

Примеры файлов OLAMIP (/examples)

Конвейеры непрерывной интеграции (/.github/workflows)

Начало работы
См. docs/overview.md для общего введения.
