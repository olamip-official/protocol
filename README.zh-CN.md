开放语言对齐可机器解释协议（OLAMIP）定义了一种开放、可互操作的文件格式，用于以结构化、可机器读取的方式表示语言学、形态学和音系学数据。

本仓库包含：

人类可读文档（/docs）

机器可读模式文件（/specs）

OLAMIP 示例文件（/examples）

持续集成工作流（/.github/workflows）

入门指南
请参阅 docs/overview.md 获取高层次介绍。
