オープン言語整合機械可読プロトコル（OLAMIP）は、言語学・形態論・音韻論データを、構造化された機械可読形式で表現するための、オープンで相互運用可能なファイル形式を定義します。

このリポジトリには以下が含まれます：

人間が読めるドキュメント（/docs）

機械可読スキーマ（/specs）

OLAMIP のサンプルファイル（/examples）

継続的インテグレーションのワークフロー（/.github/workflows）

はじめに
概要については docs/overview.md を参照してください。
