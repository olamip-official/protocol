The Open Language‑Aligned Machine‑Interpretable Protocol (OLAMIP) defines an open, interoperable file format for representing linguistic, morphological, and phonological data in a structured, machine‑readable way.

This repository includes:

- Human‑readable documentation (`/docs`)  
- Machine‑readable schemas (`/specs`)  
- Example OLAMIP files (`/examples`)  
- Continuous integration workflows (`/.github/workflows`)  

## Getting Started

See `docs/overview.md` for a high‑level introduction.
