# OLAMIP Roadmap

*Copyright © 2025 Ralph Gonzalez – https://olamip.org*

## v1.0.0 – Core Specification

- Finalize core JSON schema for `olamip.json`.  
- Publish minimal reference implementation.  
- Add basic examples for common website types (blog, ecommerce, docs).

## v2.0.0 – Enhanced Features and Tools

- Extend metadata fields (e.g., richer policy, ownership, provenance).  
- Add more flexible policy and ingestion controls.  
- Release a CLI validation tool to check schema and format compliance.

## Future Direction

- Build a web‑based validator for interactive use.  
- Develop a visual schema explorer for the OLAMIP specification.  
- Add more example configurations and CMS integrations as adoption grows.
